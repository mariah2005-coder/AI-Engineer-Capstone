# AI-Engineer-Capstone

My work for the AI Engineer internship capstone project, built with AI-assisted development tools like Claude Code.

## About

This repository tracks the capstone deliverables for the AI Engineer internship: design, implementation, and documentation of an AI-focused engineering project developed with AI pair-programming workflows.

## Teck Stack

- **Language:** JavaScript / TypeScript
- **Tools:** Cursor

## Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (LTS recommended)
- npm or another Node package manager

### Setup

```bash
git clone <repository-url>
cd AI-Engineer-Capstone
```

Install dependencies and run the project once those scripts are defined:

```bash
npm install
npm run dev
```

> **Note:** This repository is in early development. Replace `<repository-url>` with your clone URL and update the commands above when `package.json` is added.
